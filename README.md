 # Void Digital Assistant

A lightweight Android utility designed to **eliminate intrusive digital assistant pop-ups** caused by faulty or unstable headphone connections.

---

## 📱 Overview

**Void Digital Assistant** solves a frustrating issue where apps like **Google Gemini** (or other assistants) repeatedly trigger overlays due to:

* Faulty headphone jack
* Loose wired earphones
* Frequent connect/disconnect signals
* Bluetooth instability

These pop-ups can:

* Freeze your screen
* Interrupt gameplay 🎮
* Disrupt your workflow

👉 This app prevents that by **taking control as the default digital assistant** and **blocking unwanted overlays**.

---

## ✨ Features

* 🚫 Stops repeated assistant pop-ups
* 🎮 Prevents game freezes caused by overlays
* ⚡ Lightweight and fast
* 🔒 No unnecessary permissions
* 🛠 Simple setup (set as default assistant)

---

## ⚙️ How It Works

Android allows only **one default digital assistant app**.

Void Digital Assistant:

1. Replaces the default assistant (e.g., Gemini)
2. Intercepts assistant triggers
3. Prevents overlay pop-ups from appearing

## 🛠 Installation & Setup

1. Download the APK
2. Install it on your Android device
3. Go to:

   * **Settings → Apps → Default Apps → Digital Assistant App**
4. Select **Void Digital Assistant**

✅ Done! No more annoying pop-ups.

---

## ⚠️ Important Notes

* This app **does not provide assistant features**
* Its sole purpose is to **block unwanted assistant triggers**
* Works best for users experiencing headphone-related issues

---

## 🧩 Troubleshooting

If something is not working as expected:
